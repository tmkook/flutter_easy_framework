## Getting started

add pubspec.yaml
```
dependencies:
  flutter_easy_framework:
    git:
      url: git@github.com:tmkook/flutter_easy_framework.git
      path: plugins

assets
  apps/assets/
```

run commands
```
flutter pub get
flutter pub run flutter_easy_framework
```
## File structure
```
 - main.dart
 - apps your Application code
   - assets Static files
   - logics Controllers
   - states Models
   - views Pages
   bootstrap.dart Execute before running
   routes.dart Pages routes
```
